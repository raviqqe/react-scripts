# @raviqqe/react-scripts

Fork of [`react-scripts`](https://github.com/facebook/create-react-app/tree/master/packages/react-scripts).

## Install

```
npm install -D @raviqqe/react-scripts
```

## Usage

```
react-scripts init
react-scripts build
react-scripts watch
react-scripts lint
react-scripts test
```
